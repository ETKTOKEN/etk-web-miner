# ETK Web Mining Widget

The official **ETK (E-Talk Webchain)** mining widget.  
This allows any partner website to integrate ETK mining by simply copying an embed code.

---

## 🚀 Features
- Fully branded with ETK logo
- Works cross-site with API + Site Key
- Secure configuration bound to domain
- Users log in with their **e-talk.xyz account**
- Mining rewards distributed via **staking logic** (370 ETK = validator, rewards every 60 min)

---

## 📥 Installation

1. Copy the **embed code** from `embed-snippet.html`
2. Paste it into your website’s HTML
3. Configure your **API key, Secret, and Site ID** in the admin panel
4. Done! Your visitors can start mining ETK.

---

## 🔒 Security
- Each partner site is validated via **domain + Site ID**
- Configuration is protected by a password (per site, once only)
